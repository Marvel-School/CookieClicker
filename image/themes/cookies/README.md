# Cookie Skins

Place cookie skin images here. Each image should be 250x250px for best results.

## Naming Convention:
- classic.png - The default cookie
- dark.png - Dark theme cookie
- neon.png - Neon theme cookie
- [custom-name].png - Custom cookie skins

## Properties:
- Image format: PNG with transparency
- Dimensions: 250x250px recommended
- File size: Keep under 100KB for performance
